# Student Performance Prediction

This project explores how machine learning can be used to predict student performance based on academic, behavioral, and demographic data.

---

## Project Overview

The main goal of this project is to build and evaluate models that can predict students' final grades (G3).

The workflow includes:
- Data exploration (EDA)
- Feature selection
- Model training
- Hyperparameter tuning
- Error analysis

---

## Models Used

- Decision Tree Regressor  
- Random Forest Regressor  

Random Forest was the best-performing model in this project.

---

## Techniques Applied

- GridSearchCV for hyperparameter tuning  
- Cross-validation for model evaluation  
- Feature importance analysis  
- Error analysis  
- Feature selection and reduction  

---

## Key Findings

- Variables like **failures**, **absences**, and **studytime** had the strongest impact on performance  
- Random Forest performed better than Decision Tree due to better generalization  
- The model performs well on average, but struggles with extreme cases  
- A reduced set of features (around 4–5) still gives good results
